# AMS1117
Low-dropout linear regulator family.
## Pin Map
SOT-223 pinout commonly GND/ADJ, OUT, IN with tab tied to OUT.
## Absolute Maximums
Thermal dissipation is limited; respect dropout and input max.
## Typical Use
3.3 V rails for microcontroller modules.
## Common Faults
Missing output capacitor ESR requirement, tab shorted unexpectedly, overheating.
