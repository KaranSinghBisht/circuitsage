# BC557
Small-signal PNP BJT used for low-current switching, current mirrors, and complementary amplifier examples.

## Pin Map
TO-92 pinout is commonly collector, base, emitter when viewed from the flat face, but vendor variants exist. Verify the exact package before inserting into a breadboard.

## Absolute Maximums
Respect collector-emitter voltage, collector current, base-emitter reverse voltage, and package power. Limit base current with a resistor.

## Typical Use
High-side low-current switching, complementary push-pull demonstrations, bias networks, and mirrored counterpart experiments with BC547-style NPN devices.

## Common Faults
Emitter and collector swapped, PNP polarity treated like an NPN transistor, base resistor omitted, base drive referenced to the wrong rail, and load placed on the wrong side of the transistor.
