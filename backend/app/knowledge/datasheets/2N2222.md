# 2N2222
NPN small-signal switching transistor.
## Pin Map
TO-92 pinout varies by vendor; verify emitter, base, collector from package datasheet.
## Absolute Maximums
Collector current and Vce limits depend on variant; base-emitter reverse voltage is small.
## Typical Use
Low-side LED/relay drivers and common-emitter amplifiers.
## Common Faults
Emitter/collector swapped, base resistor missing, saturated when biased as amplifier.
