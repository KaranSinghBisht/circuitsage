# 7905
Fixed negative 5 V linear regulator for small analog rails and split-supply lab circuits.

## Pin Map
TO-220 front view varies by vendor, but common 79xx packages use ground, input, output ordering that differs from 78xx regulators. Verify the exact package before wiring.

## Absolute Maximums
Input must be more negative than output by the required dropout margin. Respect package dissipation, heat sinking, and capacitor voltage ratings. Reverse polarity can destroy the regulator.

## Typical Use
Generating -5 V from a negative supply, biasing op-amp experiments, and creating small negative rails for analog front ends.

## Common Faults
7805 pinout assumed for a 7905, missing input/output capacitors, no heat sink under load, input supply not negative enough, and output measured against the wrong ground reference.
